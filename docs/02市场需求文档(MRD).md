# 需求文档

## 行业领域

民航机场

### **愿景与目标**

+ 开发一款ChatBI（基于自然语言的智能商业分析）应用，让不具备 SQL 或复杂报表工具使用能力的业务人员，也能通过对话快速获取数据洞察(查询数据或者生成报表)。
+ 构建知识库，与机场私有、实时、高专业性的业务数据“挂钩”
  + 让 ChatBI应用能理解专业术语，即使用户问题中包含专业术语 ChatBI 应用能准确理解用户意图
  + 让用户能通过对话检索机场规章制度和机场静态数据
+ 单次会话中能记住上下文信息
+ 能创建新对话,能保存历史对话,能删除对话
+ 能收藏常用的问题, 点击常用问题能再次查询数据或生成图标
+ 能上传 Excel/CSV 文件,并选择上传文件,根据上传文件的数据进行问数
+ 不同部门间的只能查询部门权限内的数据,禁止越权/提权查询

### **使用者**

机场信息科技部，初期的使用者大概在5~10人之间

### **数据资源**

+ 分类：航班数据、机场资源数据、天气数据、会员数据、会员订单数据、店铺数据
+ 数量：41个表

### **常见问题**

+ 我想查询航班号CA1519在2025年6月1日的详细信息、
+ 我想查看每位会员在最近一年内的消费总金额和消费次数统计
+ 请帮我找出2月份过生日的所有白名单会员
+ 请统计过去30天内有过消费记录的活跃会员一共有多少位
+ 请列出广东省当前所有正在生效的红色和橙色级别的天气预警信息

### **常见统计**

+ 航班数据统计（5个）
  + 各航空公司航班量和准点率排名 - 分析航空公司的运营表现
  + 延误航班主要原因分析 - 找出延误原因及平均延误时长
  + 各航站楼航班量统计 - 了解航站楼繁忙度分布
  + 国内国际航班比例 - 分析航线结构
  + 热门目的地城市TOP10 - 发现最受欢迎的航线
+ 会员商业数据统计（4个）
  + VIP会员消费金额TOP100 - 识别高价值客户
  + 店铺销售额排名 - 分析商业表现
  + 会员年龄段分布 - 了解客户群体结构
  + 会员消费时段偏好 - 分析消费行为习惯
