# 业务场景

* 第一个迭代主要针对数据交换系统当前已接入的数据，实现查询与分析功能；至于其他领域的数据，就要看上线之后的实际效果、推广程度、和信息科技部领导重视程度，才能推动后续其他领域的数据接入工作。

* ChatBI使用方是首都机场信息科技部，初期的使用者大概在5~10人之间；
* 第一期主要查询与分析的数据都已接入数据交换系统，暂不涉及数据权限的问题；如果上线之后，有其它机场部门需要使用，也是会把数据先接入到数据交换系统，再进行查询分析，这时可能会涉及数据权限问题，到时候需要和信息科技部负责人确认
